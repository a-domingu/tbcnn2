Welcome to Discern Framework Project

The Discern Framework Project is an artificial intelligence software project aimed to detect software patterns in source code. It is applied to complete software projects. It is based on a TBCNN (Tree Based Convolutional Neural Network), and it is developed in the European Industrial Center at Capgemini, Spain.

Detecting a Software Design Pattern in a Python file

The Discern Framework Project is meant to receive different Python files, and use a trained Convolutional Neural Network (CNN) to detect patterns.

  • The user can upload a multiple Python files (complete software projects) compressed in a Zip file.
  • The user can decide on which pattern he wants to do the analysis.
  • The output that the user will receive is a message. The message will specify for each file if it has or does not have the required pattern.

Detecting a Software Design Pattern in a Github repository

The Discern Framework Project is meant to receive different Python files, and use a trained Convolutional Neural Network (CNN) to detect patterns.

  • The user can enter a URL of a GitHub repository to analyse.
  • The user can decide on which pattern he wants to do the analysis.
  • The output that the user will receive is a message. The message will specify for each file if it has or does not have the required pattern.

Train a TBCNN based on a prepared training data set

The TBCNN (Tree Based Convolutional Neural Network), which is a specific type of CNN that works on Abstract Syntax Tree (AST), was trained based on a prepared and labeled dataset.

The TBCNN has two independent networks:

  • The first neural network does the embedding and the vector representation for each AST.
  • The second neural network apply several transformations to train the Convolutional Neural Network (CNN).
